What You'll Learn

Deep dives into every corner of Claude Code.

Getting Started

Install Claude Code, authenticate, and run your first session. Works as a CLI, VS Code extension, desktop app, web app, and JetBrains plugin.

npm install -g @anthropic-ai/claude-code

claude to start interactive mode

claude "query" to start with a prompt

FOUNDATION

Memory & Instructions

CLAUDE.md files, auto memory, .claude/rules/, and scoped instructions that persist across sessions.

Prompting Techniques

Write effective prompts, reference files with @path, use Plan Mode, and master the explore-plan-implement flow.

Hooks & Configuration

Automate actions with lifecycle hooks (command, HTTP, prompt, agent types). Configure permissions, settings files, environment variables, and status lines across user, project, and local scopes.

MCP Servers

Connect external tools via the Model Context Protocol. Add servers with claude mcp add for databases, APIs, and more.

Skills & Plugins

Create custom slash commands with SKILL.md files, install plugins from marketplaces, and extend Claude's capabilities.

Subagents & Teams

Define custom agents in .claude/agents/, delegate tasks with worktree isolation, and run parallel agent teams.

Automation & CI/CD

Run non-interactive with claude -p, integrate into GitHub Actions, schedule recurring tasks, and build pipelines.

Essential Commands

claude -r "name"Resume by session name
claude --modelSwitch model (opus, sonnet)

Power User Tip #1
Context Management

Power User Workflows

Patterns that multiply your productivity.

Explore → Plan → Implement

1

Enter Plan Mode

Press Shift+Tab to switch to plan mode

2

Explore the Codebase

Ask Claude to read files, find patterns, understand architecture

3

Review the Plan

Claude writes a plan file. Iterate until aligned.

4

Switch to Normal Mode & Execute

Approve the plan, Claude implements it step by step

Writer / Reviewer Pattern

1

Session A writes the code

First Claude session implements the feature

2

Session B reviews it

Second session reviews with fresh context, finds issues

3

Iterate until clean

Separate contexts prevent self-reinforcing blind spots

Subagent Investigation

1

Main session delegates research

Claude spawns an Explore subagent for codebase investigation

2

Subagent explores in isolation

Separate context reads files without cluttering your main session

3

Summary returns to main session

Only the findings come back, keeping your context lean

CI/CD Automation

1

Pipe data into Claude

cat logs.txt | claude -p "analyze errors"

2

Use in GitHub Actions

Automated code review, PR descriptions, release notes

3

Schedule recurring tasks

Use /schedule or /loop for repeated operations

Configuration Cheat Sheet

Where everything lives on disk.

Instruction Files

~/.claude/CLAUDE.md Personal instructions (all projects)
./CLAUDE.md Project instructions (shared via git)
./CLAUDE.local.md Personal project notes (gitignored)
.claude/rules/*.md Scoped rules (loaded by path match)

Settings Files

~/.claude/settings.json User settings (global)
.claude/settings.json Project settings (commit to git)
.claude/settings.local.json Personal project settings (gitignored)
Load order: user → project → local (later overrides earlier)

Skills, Agents & Plugins

.claude/skills/*/SKILL.md Project-scoped custom skills
~/.claude/skills/*/SKILL.md Global custom skills
.claude/agents/*.md Custom subagent definitions
.mcp.json MCP server configuration

Memory & Session Data

~/.claude/projects/*/memory/ Auto memory per project
~/.claude/plans/ Plan mode plan files
~/.claude/sessions/ Session transcripts
~/.claude/hooks/ Hook scripts

Use subagents for investigation. They explore in a separate context, keeping your main conversation clean for implementation.

Power User Tip #2 — Context Hygiene

Permission Modes

Control how much autonomy Claude gets. Cycle with Shift+Tab.

Default

Asks before writes, edits, and most bash commands

Accept Edits

Auto-approves file edits, still asks for bash

Auto

AI-driven classifier decides which tools to auto-approve

Bypass

No prompts at all. Use with caution.
